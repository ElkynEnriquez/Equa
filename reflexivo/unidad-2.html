Jerarquía de operaciones

Es el orden en el que deben realizarse las operaciones aritméticas básicas.

Cuando una expresión aritmética involucra sumas, restas, multiplicaciones y/o divisiones.

Ejemplo:

(x + 4)2 + (x + 3)(x – 4)
= (x2 + 8x + 16) + (x + 3)(x - 4)
= x2 + 8x + 16 + x2 - 4x + 3x - 12
