Unidad 4: Operaciones con polinomios

Polinomio es una expresión algebraica constituida por una suma finita de productos entre variables y constantes, o bien una sola variable

Estilo teórico

Polinomios de una sóla variable

* Es la expresión algebraica más sencilla de utilizar

* Un polinomio de una sola variable es aquel en el que puedes ver una sola letra, aunque ésta figure más de una vez incluso elevado a algún exponente

* Se puede ordenar de forma ascende o descendente

* Para determinar su grado se evalua el mayor exponente de la parte literal

P(x)= axn – xm + bx + x – c

Ejemplo: Ordenar el polinomio de forma ascendente

P(x) = 3x2 – 4x – 5x3 + 4
P(x) = 4 – 4x + 3x2 – 5x3

Binomios

* Están formadas sólo por dos términos que se separan por un signo de suma (+) o de resta (-).

* Se compone de la suma de dos monomios.

2a2 + 3b3

* El grado de un binomio es el exponente unido al término exponente unido al termino.

4x2

Binomio de Newton

Existe una fórmula que permite calcular las potencias de ese binomio, con una expresión de n+1 términos

(a + b)2 = a2 + 2ab + b2

(a - b)2 = a2 - 2ab - b2

Ejemplo:

(x + 1)2
= x2 + 12 + 2 * x * 1
x2 + 2x + 1

Operaciones con polinomios

Adición de polinomios:

Primero escribimos un polinomio y seguido en la misma línea escribimos el otro que vamos a sumar o restar. Después, agrupamos términos semejantes.

P(x) + Q(x)

Ejemplo:

P(x) + Q(x) = (4x2 – 12xy + 9y2) + (25x2 + 4xy – 32y2)
P(x) + Q(x) = (4x2 + (– 12xy) + 9y2 + 25x2 + 4xy + (– 32y2)
P(x) + Q(x) = (4x2 + (– 12xy) + 9y2 + 25x2 + 4xy + (– 32y2)
P(x) + Q(x) = 29x2 + (−8xy) +(−23y2)
P(x) + Q(x) = 29x2 – 8xy – 23y2

Sustracción de polinomios:

Restar polinomios, se puede decir que es lo mismo que sumar polinomios , sólo tenemos que cambiar el segundo polinomio llamado sustraendo por su opuesto, aplicando la ley de los signos.

P(x) - Q(x)

De la resta P(x) - Q(x) donde Q(x) = 2x3 - 3x2 + x es el sustraendo, su opuesto sería -Q(x) = -2x3 + 3x2 - x

Finalmente al minuendo debemos sumarle el sustraendo obteniendo una suma de polinomios

P(x) - {-Q(x)s

Ejemplo:

P(x) − Q(x) = (4y³ + 4y − 2) − (2y³ − 3y² + 4y)
P(x) − Q(x) = 4y³ + 4y − 2 − 2y³ + 3y² − 4y
P(x) − Q(x) = 4y³ − 2y³ + 3y² + 5y − 4y − 2
P(x) − Q(x) = 2y³ - 3y² + y − 2

Multiplicación de polinomios:

Se multiplica cada monomio del primer polinomio por todos los elementos del segundo polinomio.

P(x) * Q(x)

Se suman los monomios del mismo grado

Ejemplo:

P(x) * Q(x) = (5x² − 4) * (5x³− 2x² + x)
P(x) * Q(x) = 25x5 − 10x4 + 5x³− 20x³+ 8x²− 4x
P(x) * Q(x) = 25x5 − 10x4 + 15x³+ 8x²− 4x
Grado del polinomio = Grado de P(x) + Grado de Q(x) = 2 + 3 = 5

División de polinomios:

P(x) / Q(x)
